# 📱 WhatsApp Bulk Automation Tool

A professional-grade Python automation tool to send bulk WhatsApp messages using Selenium and Excel. This tool features a modern GUI, multi-threading for smooth performance, and dynamic user inputs.

---

## 📂 Project Structure

* **main.py** – The core Python script containing the GUI logic (CustomTkinter) and Selenium automation.
* **requirements.txt** – List of necessary Python libraries (Selenium, CustomTkinter, Openpyxl).

---

## 🚀 How to Run

### # Clone the repository
```bash
git clone https://github.com/Ashishpatel27/whatsapp-automation-tool.git

# Navigate into the project
cd whatsapp-automation-tool

# Install dependencies
pip install customtkinter selenium openpyxl

# Start the application
python main.py
